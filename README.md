# CSFCP - Portfolio Mini Project

## Description
This is a simple, clean, and Portfolio Website created as part of the Computer Science Fundamentals (CSF) Mini Project. It showcases personal information, projects, and development skills using HTML and CSS. The project also demonstrates the use of proper documentation, commit history, screenshots, and recommended project structure.

## Features
* Clean and professional landing page
* "About Me" section
* "My Projects" section
* Simple and responsive layout
* Developed using pure HTML & CSS
* Organized folder structure
* Includes documentation and screenshots
* Version-controlled using Git

## Technologies Used
* HTML5
* CSS3
* Git & GitHub
* VS Code

## Installation and Usage Instructions
Since this is a static website, you don't need to install complex software. Follow these steps:

1. **Clone the Repository:**
   Open your terminal and run:
   ```bash
   git clone https://github.com/Rishika-bhambri/CSFCP.git
   ```
2. **Open the Project:** Navigate to the project folder:

```bash
cd CSFCP
```

3. **Run the Website:**

* Locate the index2.html file in the folder.

* Double-click it to open it in your web browser (Chrome, Edge, etc.).
